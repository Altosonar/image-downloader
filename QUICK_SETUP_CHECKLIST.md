# Quick Setup Checklist

## 🚀 Install Enhanced Image Downloader

### Step 1: Load Extension in Chrome
- [ ] Open Chrome Extensions page (chrome://extensions/)
- [ ] Enable "Developer mode" (toggle in top right)
- [ ] Click "Load unpacked" 
- [ ] Navigate to and select the `build` folder
- [ ] Extension should appear in your extensions list

### Step 2: Test Smart Features
- [ ] Visit any website with images (try: https://unsplash.com)
- [ ] Click the extension icon (should show image icon)
- [ ] Hover over images in the popup to see smart titles
- [ ] Click images to select/deselect them
- [ ] Enable "Show file renaming" in options if not already enabled
- [ ] Download selected images to see smart naming

### Step 3: Verify Functionality
- [ ] ✅ Images appear in popup
- [ ] ✅ Hover shows smart titles
- [ ] ✅ Click to select/deselect works
- [ ] ✅ Filter by URL works
- [ ] ✅ Download button works
- [ ] ✅ Files are named using smart titles

### Step 4: Test with Demo Pages
- [ ] Open `test-smart-titles.html` in browser
- [ ] Load extension on this page
- [ ] Test all image types and scenarios

### Troubleshooting
- [ ] If extension doesn't load, check all files are in build folder
- [ ] If features don't work, reload the extension
- [ ] Check Chrome Developer Tools for any errors
- [ ] Ensure you're on a page with images

### Success Indicators
- [ ] Smart title badge shows "SMART TITLES ACTIVE"
- [ ] Hover overlays appear on images
- [ ] Downloaded files have meaningful names
- [ ] All buttons and filters respond correctly

**Once loaded, the extension will be fully functional with all smart features!**
